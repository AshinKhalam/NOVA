UIDC (User Interface Designing Contest)

**intro**

About The Event

Create a layout for a food delivery application that demonstrates what interface elements will exist on key pages. This app should permit users to place food orders from any restaurant of their choice in any area using their smartphone device. The food delivery is done at the doorstep of the customers with no hassle. Customers can choose any payment mode to pay for their orders.

Instructions

Participants should create the best version of their interface of the application of given category.

Participants should try out interesting techniques and show their personality in their work.

Only submit one file for this contest.

Participants are requested to submit their respective entries in provided google forms in pdf format. (maximum file size: 10MB)

Participants should make sure that the source file is shared with us and should be unique and their own.

Can be done using pen and paper or digitally using apps such as canva (refer to the video below)
